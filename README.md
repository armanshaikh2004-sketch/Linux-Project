 Linux System Administration Lab Project

 Overview
This project demonstrates core Linux system administration tasks including storage management, backups, web server deployment, and file permissions.

 1. LVM (Logical Volume Management)
- Created Physical Volumes, Volume Groups, and Logical Volumes
- Extended and reduced logical volumes

 2. Backup Automation (Cron)
- Configured scheduled backup jobs using cron
- Automated system backup process

 3. Web Server (Apache)
- Installed and configured Apache web server
- Enabled firewall rules for HTTP access
- Hosted basic web page

 Tools Used
- Linux (Ubuntu/CentOS)
- Apache Web Server
- Cron
- LVM utilities

 Outcome
Gained hands-on experience in Linux system administration and server management.
